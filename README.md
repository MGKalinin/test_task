# Задача:   
1. Использовать сваггер как документацию, на основе него написать автотесты, покрывающие функциональные требования.    
Из инструментов использовать pytest+allure(библиотека allure-pytest).

# Инструкция по использованию:   
## 1.Склонировать репозиторий на свой компьютер
## 2.Установить зависимости:   
> ```bash
> pip install -r requirements.txt
> ```   

## 3.Добавить файл .env в котором прописать адрес BASE_URL , например:   
```BASE_URL=http://google.com```   

## 4.Запуск pytest с подробным выводом
>```bash
> pytest -v
>```   


## 5.Запуск pytest с allure 
 
### Шаг A: Удалить предыдущие отчёты- если существуют - иначе переходить к шагу B
>```bash
> rm -rf ./allure-results/*
>```   
### Шаг B: Запуск тестов  
> ```bash
> pytest tests/ --alluredir=./allure-results
>```   

### Шаг C: Просмотр отчёта (нужно allure-cli установить в систему (ОС))
>```bash
> allure serve ./allure-results
>```

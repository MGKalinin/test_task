# Задача:   
1. Использовать сваггер как документацию, на основе него написать автотесты, покрывающие функциональные требования.    
Из инструментов использовать pytest+allure(библиотека allure-pytest).

# Инструкция по использованию
## Склонировать репозиторий на свой компьютер
## Установить зависимости:   
> ```bash
> pip install -r requirements.txt
> ```   

## Добавить файл .env в котором прописать адрес BASE_URL , например:   
```BASE_URL=http://google.com```   

## Запуск pytest с подробным выводом
>```bash
> pytest -v
>```   


## Запуск pytest с allure 
 
### Шаг 1: Удалить предыдущие отчёты- если существуют - иначе переходить к шагу 2
>```bash
> rm -rf ./allure-results/*
>```   
### Шаг 2: Запуск тестов  
> ```bash
> pytest tests/ --alluredir=./allure-results
>```   

### Шаг 3: Просмотр отчёта (нужно allure-cli установить в систему (ОС))
>```bash
> allure serve ./allure-results
>```
